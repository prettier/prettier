#### Fix line breaks when css attribute values contain literal newlines (#18605 by @kovsu)

<!-- prettier-ignore -->
```css
/* Input printWidth: 30 */
div span[foo="bar\
baz-baz-baz"] {
  color: red;
}

/* Prettier stable */
div
  span[foo="bar\
baz-baz-baz"] {
  color: red;
}

/* Prettier main */
div span[foo="bar\
baz-baz-baz"] {
  color: red;
}
```
