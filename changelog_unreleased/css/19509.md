#### Preserve CSS `attr()` type multipliers (#19509 by @1cbyc)

<!-- prettier-ignore -->
```css
/* Input */
div {
  border-radius: attr(data-size type(<length> +));
}

/* Prettier stable */
div {
  border-radius: attr(data-size type(<length> +));
}

/* Prettier main */
div {
  border-radius: attr(data-size type(<length>+));
}
```
