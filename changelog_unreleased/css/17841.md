#### Handle attribute selector with case-sensitive and uppercase case-insensitive flags (#17841, #17865 by @kovsu)

<!-- prettier-ignore -->
```css
/* Input */
[type=a s],
[type=a S],
[type=a I] {
  list-style-type: lower-alpha;
}

/* Prettier stable */
[type="a s"],
[type="a S"],
[type="a I"] {
  list-style-type: lower-alpha;
}

/* Prettier main */
[type="a" s],
[type="a" S],
[type="a" I] {
  list-style-type: lower-alpha;
}
```
