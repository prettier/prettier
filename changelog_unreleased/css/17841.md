#### Handle attribute selector with case-sensitive flag (#17841 by @kovsu)

<!-- prettier-ignore -->
```css
/* Input */
ol[type=a s] {
  list-style-type: lower-alpha;
}

/* Prettier stable */
ol[type="a s"] {
  list-style-type: lower-alpha;
}

/* Prettier main */
ol[type="a" s] {
  list-style-type: lower-alpha;
}
```
