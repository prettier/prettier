#### Preserve quote style for embedded HTML attribute values (#18352 by @kovsu)

This also affects lit-html template literals.

<!-- prettier-ignore -->
```html
// Input
<div foo="{data}" bar='{data}' baz={data}></div>

// Prettier stable
<div foo={data} bar={data} baz={data}></div>

// Prettier main
<div foo="{data}" bar="{data}" baz={data}></div>
```
