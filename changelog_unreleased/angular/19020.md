#### Angular control flow blocks now respect `--html-whitespace-sensitivity` (#19020 by @mattlewis92)

Previously, Angular control flow blocks (`@if`, `@else`, `@for`, `@switch`, `@defer`) would always have whitespace added around and inside their content when formatted, even inside inline elements where this changes the rendered output.

<!-- prettier-ignore -->
```html
<!-- Input -->
<span>Foo@if(true) {bar}</span>baz

<!-- Prettier stable -->
<span>
  Foo
  @if (true) {
    bar
  }
</span>
baz

<!-- Prettier main -->
<span>Foo@if (true) {bar}</span>baz
```
