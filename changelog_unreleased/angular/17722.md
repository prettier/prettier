#### Support Angular 21 (#17722, #18294 by @fisker)

Angular 20.1 added support for new assignment operators.
Angular 21 added support for regular expression.

<!-- prettier-ignore -->
```html
<!-- Input -->
<b (click)="
  a   ??= b">{{  /\d+/g}}</b>

<!-- Prettier stable -->
<b
  (click)="
  a   ??= b"
  >{{  /\d+/g}}</b
>

<!-- Prettier main -->
<b (click)="a ??= b">{{ /\d+/g }}</b>
```
