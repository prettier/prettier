#### Ignore comments inside mapped type when checking type parameter comments (#19572 by @fisker)

<!-- prettier-ignore -->
```tsx
// Input
foo<{
  // comment
  [key in keyof Foo]: number
}>();

// Prettier stable
foo<
  {
    // comment
    [key in keyof Foo]: number;
  }
>();

// Prettier main
foo<{
  // comment
  [key in keyof Foo]: number;
}>();
```
