#### Fix formatting for `NewExpression` with type cast (#18406)

Previously, `NewExpression` with a type cast was formatted inconsistently compared to `CallExpression`.

```ts
// Input
new (require("./webpack/plugins/next-trace-entrypoints-plugin")
  .TraceEntryPointsPlugin as P)({
  rootDir: dir,
});

// Prettier stable
new (require("./webpack/plugins/next-trace-entrypoints-plugin")
  .TraceEntryPointsPlugin as P)({
  rootDir: dir,
});

// Prettier main
new (require("./webpack/plugins/next-trace-entrypoints-plugin")
  .TraceEntryPointsPlugin as P)({
  rootDir: dir,
});
