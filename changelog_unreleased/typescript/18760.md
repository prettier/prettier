#### Preserve parentheses for conditional types in type parameter constraint (#18760 by @kovsu)

<!-- prettier-ignore -->
```ts
// Input
<Foo extends (Bar extends Baz ? A : B)>() => true;

// Prettier stable
<Foo extends Bar extends Baz ? A : B>() => true;

// Prettier main
<Foo extends (Bar extends Baz ? A : B)>() => true;
```
