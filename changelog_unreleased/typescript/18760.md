#### Add parentheses to conditional types in type parameter constraint (#18760 by @kovsu)

<!-- prettier-ignore -->
```ts
// Input
const foo = <Foo extends (Bar extends Baz ? A : B)>() => true;

// Prettier stable
const foo = <Foo extends Bar extends Baz ? A : B>() => true;

// Prettier main
const foo = <Foo extends (Bar extends Baz ? A : B)>() => true;
```
