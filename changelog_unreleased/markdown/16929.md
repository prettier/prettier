#### Fix inconsistent alignment of multi-paragraph list items (#16929 by @mldangelo)

Subsequent paragraphs in list items now align with the first paragraph. The first item's spacing determines the style for the entire list.

<!-- prettier-ignore -->
```md
<!-- Input -->
-   First paragraph.

    Second paragraph.

<!-- Prettier stable -->
- First paragraph.

    Second paragraph.

<!-- Prettier main -->
-   First paragraph.

    Second paragraph.
```
