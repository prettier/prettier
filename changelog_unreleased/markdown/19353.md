#### Preserve original fence length of fenced code blocks (#19353 by @WhoamiI00)

When the source uses more backticks (or tildes) than the strict minimum needed to escape the block's content, Prettier now keeps that original length instead of shrinking it. This is particularly useful for nested code fences (e.g. MyST directives) where longer fences are picked deliberately to visually distinguish nesting levels.
