#### Support directives in markdown parser (#19662 by @Cardoso222)

Prettier no longer destroys directives (used by Docusaurus admonitions, among
others) when formatting markdown. Previously, with `proseWrap: "always"`, the
directive fences were merged into the surrounding text.

<!-- prettier-ignore -->
```md
<!-- Input -->
:::note
Some note
:::

<!-- Prettier stable (--prose-wrap=always) -->
:::note Some note :::

<!-- Prettier main (--prose-wrap=always) -->
:::note
Some note
:::
```
