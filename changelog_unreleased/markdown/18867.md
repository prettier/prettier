#### Fix CJK line break loss near list-like numbers (#18867 by @it-education-md)

Avoid collapsing line breaks before list-like numbers followed by CJK characters (e.g. 3.中).

<!-- prettier-ignore -->
```markdown
<!-- Input -->
1.a
2.b
3.中

<!-- Prettier stable -->
1.a
2.b 3.中

<!-- Prettier main -->
1.a
2.b
3.中
```
