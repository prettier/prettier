#### Avoid corrupting empty link with title (#19487 by @andersk)

Do not remove `<>` from an inline link or image with an empty URL and a title, as this removal would change its interpretation.

<!-- prettier-ignore -->
```md
<!-- Input -->
[link](<> "title")

<!-- Prettier stable -->
[link]( "title")

<!-- Prettier main -->
[link](<> "title")
```
