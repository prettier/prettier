#### Support range formatting for YAML (#19299 by @seirl)

Support range formatting for YAML files. Previously, requesting range
formatting on a YAML file would return the file unchanged (format nothing).
Now, it correctly formats only the selected mapping and sequence items.

<!-- prettier-ignore -->
```yaml
# Input (selected range: lines 2-3)
a:      1
b:  2
c:   3
d:      4

# Prettier stable
a:      1
b:  2
c:   3
d:      4

# Prettier main
a:      1
b: 2
c: 3
d:      4
```
