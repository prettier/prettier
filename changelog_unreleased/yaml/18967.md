#### Use block syntax for multi-line flow collections, and support `objectWrap: collapse` (#18967 by @pauldraper)

Multi-line flow collections (mappings and sequences) are now converted to block syntax.

<!-- prettier-ignore -->
```yaml
# Input
key: {a: 1,
  b: 2}

# Prettier stable
key: { a: 1, b: 2 }

# Prettier main
key:
  a: 1
  b: 2
```

With `objectWrap: collapse`, any collection is rewritten as flow syntax if it fits on a single line.

<!-- prettier-ignore -->
```yaml
# Input
block_mapping:
  a: 1
  b: 2

# Prettier stable (--object-wrap=collapse)
block_mapping:
  a: 1
  b: 2

# Prettier main (--object-wrap=collapse)
block_mapping: { a: 1, b: 2 }
```
