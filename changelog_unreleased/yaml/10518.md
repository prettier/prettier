#### Preserve newlines between YAML anchors and comments (#10518 by @Sushanth012)

YAML comments on a later line after an anchor are no longer moved onto the anchor's line.

<!-- prettier-ignore -->
```yaml
# Input
key1: &default

  # This key ...
  subkey1: value1

# Prettier stable
key1: &default # This key ...
  subkey1: value1

# Prettier main
key1: &default
  # This key ...
  subkey1: value1
```
