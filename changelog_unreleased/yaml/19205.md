#### Fix YAML formatting: preserve empty line after block scalar (#19205 by @seirl)

Empty lines are no longer incorrectly discarded one line after a block scalar.

<!-- prettier-ignore -->
```yaml
# Input
foo: >
  a
  b
bar: baz

qux: quux

# Prettier stable
foo: >
  a
  b
bar: baz
qux: quux

# Prettier main
foo: >
  a
  b
bar: baz

qux: quux
```
