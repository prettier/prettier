#### Format empty JSX tags to self-closing tags (#19136 by @AlpianPPLG)

When a JSX element has no meaningful children (for example `<Foo></Foo>`),
Prettier now formats it as a self-closing tag (`<Foo />`) for conciseness and
consistency with common JSX style conventions.

```jsx
// Input
<Foo></Foo>;

// Prettier
<Foo />;
```
