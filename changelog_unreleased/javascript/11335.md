#### Support hack-style pipeline proposal (#11335 by @sosukesuzuki)

The proposed [hack-style pipeline syntax](https://github.com/js-choi/proposal-hack-pipes) is now supported by Prettier. We are using `%` as the topic token, following the official explainer â€” although this may change in future releases as the proposal progresses.
