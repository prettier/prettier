#### Fix range formatting for arrow functions as call arguments (#18752 by @kovsu)

<!-- prettier-ignore -->
```js
// Input
fn(
    () => {
    },
//  ^^ Range
    () => {
//  ^^^^^^^ Range
    }
);

// Prettier stable
f(
    (() => {}, () => {});,
);

// Prettier main
f(
  () => {},
  () => {},
);
```
