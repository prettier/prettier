#### Fix duplicated comments in empty branches with `experimentalTernaries` (#18963 by @kovsu)

<!-- prettier-ignore -->
```jsx
// Input
condition ? ifTrue
: [
    // Hello, world!
  ];

// Prettier stable (--experimental-ternaries)
condition ? ifTrue
  // Hello, world!
: (
  [
    // Hello, world!
  ]
);

// Prettier main (--experimental-ternaries)
condition ? ifTrue : (
  [
    // Hello, world!
  ]
);
```
