#### Support type cast comments for `acorn` and `espree` parser (#17491, #17566 by @ArnaudBarre, #17587 by #fisker)

This was previously only supported by the Babel parser.

<!-- prettier-ignore -->
```js
// Input
/** @type {MyType} */ (x).foo;

// Prettier stable (--parser=acorn or --parser=espree)
/** @type {MyType} */ x.foo;

// Prettier main
/** @type {MyType} */ (x).foo;
```
