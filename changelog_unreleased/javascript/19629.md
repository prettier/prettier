#### Added a new official plugin `@prettier/plugin-yuku` (#19628, #19629 by @fisker)

`@prettier/plugin-yuku` is powered on Yuku(A high-performance JavaScript/TypeScript compiler toolchain written in Zig).

This plugin includes two new parsers `yuku` (JavaScript syntax) and `yuku-ts` (TypeScript syntax), to use this plugin:

Install the plugin

```bash
yarn add --dev prettier @prettier/plugin-yuku
```

Add the following to your `.prettierrc` file

```yaml
plugins:
  - "@prettier/plugin-yuku"
```

Due to package size limitations, this plugin is not included in the prettier package, it needs to be installed separately.

For more information, check [the package homepage](https://github.com/prettier/prettier/tree/main/packages/plugin-yuku).
