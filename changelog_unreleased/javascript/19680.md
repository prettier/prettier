#### Avoid extra parentheses for nullish coalescing in JSX-mode ternaries (#19680 by @kushalsacharya)

<!-- prettier-ignore -->
```jsx
// Input
condition ? (
  <Example /> // comment
) : (
  "alpha" ?? "bravo"
);

// Prettier stable
condition ? (
  <Example /> // comment
) : (
  ("alpha" ?? "bravo")
);

// Prettier main
condition ? (
  <Example /> // comment
) : (
  "alpha" ?? "bravo"
);
```
