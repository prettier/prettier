#### Add support for dynamic tag names in html template literals (#15545 by @tbz)

<!-- prettier-ignore -->
```js
// Input
const tag = 'div';
html`
<${tag}>
</${tag}>
`;
html`
<${literal(tag)}>
</${literal(tag)}>
`;
```
