#### Don't force array/object with short comments to break (#18617 by @fisker)

<!-- prettier-ignore -->
```jsx
// Input
const array = [/* comment */]
const object = {/* comment */}

// Prettier stable
const array = [
  /* comment */
];
const object = {
  /* comment */
};

// Prettier main
const array = [/* comment */];
const object = {/* comment */};
```
