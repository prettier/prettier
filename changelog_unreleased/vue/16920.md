#### Support `.prop` shorthand (#16920 by @fisker)

`.foo` is shorthand for `v-bind:foo.prop`. See [`v-bind` builtin directive](https://vuejs.org/api/built-in-directives.html#v-bind) for details.

<!-- prettier-ignore -->
```vue
<!-- Input -->
<template>
  <button .disabled="   a &&b ">Click!</button>
</template>

<!-- Prettier stable -->
<template>
  <button .disabled="   a &&b ">Click!</button>
</template>

<!-- Prettier main -->
<template>
  <button .disabled="a && b">Click!</button>
</template>
```
