#### Avoid adding trailing comma to parenthesized SCSS scalars (#19091 by @cyphercodes)

<!-- prettier-ignore -->
```scss
// Input
.a { background-color: rgba($color: (#808080), $alpha: 0.9); }

// Prettier stable
.a {
  background-color: rgba(
    $color: (
      #808080,
    ),
    $alpha: 0.9
  );
}

// Prettier main
.a {
  background-color: rgba($color: (#808080), $alpha: 0.9);
}
```
