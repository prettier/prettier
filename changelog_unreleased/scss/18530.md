#### Prevent adding trailing comma in math expression within function arguments (#18530 by @kovsu)

<!-- prettier-ignore -->
```scss
// Input
@include container($foo: 2 * ($bar + $baz));

// Prettier stable
@include container(
  $foo: 2 *
    (
      $bar + $baz,
    )
);

// Prettier main
@include container($foo: 2 * ($bar + $baz));
```
