#### Remove extra spaces after `[` in map lookups (#19503 by @kovsu)

<!-- prettier-ignore -->
```less
// Input
.foo {
  color: #theme[ primary];
  color: #theme[@name];
  color: #theme[@@name];
}

// Prettier stable
.foo {
  color: #theme[ primary];
  color: #theme[ @name];
  color: #theme[ @@name];
}

// Prettier main
.foo {
  color: #theme[primary];
  color: #theme[@name];
  color: #theme[@@name];
}
```
