#### Remove extra spaces after `[` in Less map lookups (#19503 by @kovsu)

<!-- prettier-ignore -->
```less
// Input
.foo {
  color: @colors[ primary];
  color: @colors[ @name];
  color: @colors[ @@name];
}

// Prettier stable
.foo {
  color: @colors[ primary];
  color: @colors[ @name];
  color: @colors[ @@name];
}

// Prettier main
.foo {
  color: @colors[primary];
  color: @colors[@name];
  color: @colors[@@name];
}
```
