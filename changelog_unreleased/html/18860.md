#### Respect `htmlWhitespaceSensitivity: "strict"` for SVG elements (#18860 by @olagokemills)

All SVG elements now respect `htmlWhitespaceSensitivity: "strict"` to preserve whitespace, ensuring content like `<text>{{ value }}</text>` isn't broken by newlines.

<!-- prettier-ignore -->
```html
<!-- Input -->
<svg><text>a</text><text>b</text></svg>

<!-- Prettier stable -->
<svg>
  <text>a</text>
  <text>b</text>
</svg>

<!-- Prettier main -->
<svg><text>a</text><text>b</text></svg>
```
